# variables.js

#Installation
### Download the ```variable.js``` file and link the file with your html file. 

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <script src="variables.js"></script>
  </head>
  <body>
  </body>
</html>
```

```html
<body>
  ......
  ......
  ......
  <script>
    vars = new Variables("var");
    vars.fill();
  </script>
 </body>
```
